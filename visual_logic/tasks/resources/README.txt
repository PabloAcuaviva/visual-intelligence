## Conway's Game of Life Pattern Collection

This repository includes pattern files sourced from [LifeWiki](https://www.conwaylife.com/wiki/Main_Page), one of the most comprehensive collections of Game of Life patterns available.

- Source: [LifeWiki Pattern Collection](https://www.conwaylife.com/wiki/Downloadable_pattern_collections)
- Archive: 6MB .zip containing 5200+ RLE pattern files
- Includes: 1732 patterns (705 oscillators, 209 spaceships, 374 still lifes), 3643 images, and glider syntheses for 995 patterns

All content is credited to the LifeWiki community and used here under fair use for research purposes.

## Chess Mate in 123

This repository includes chess puzzle data sourced from [Hugging Face Datasets](https://huggingface.co/datasets/quantum24/chess_puzzles_10k_in_pgn_san), which provides a curated collection of checkmate-in-1, checkmate-in-2, and checkmate-in-3 problems derived from the Lichess puzzle database.

- Source: [quantum24/chess_puzzles_10k_in_pgn_san](https://huggingface.co/datasets/quantum24/chess_puzzles_10k_in_pgn_san)
- Archive: 10,000 puzzles in JSON and PGN format
- Includes: 3,333 mate-in-1 puzzles, 3,333 mate-in-2 puzzles, 3,334 mate-in-3 puzzles
- Each entry provides: FEN position, full PGN context, UCI and SAN solution moves, and thematic tags

All content is credited to the [Lichess](https://lichess.org) community puzzle database and distributed via Hugging Face under the [CC0 1.0 license](https://creativecommons.org/publicdomain/zero/1.0/), making it freely available for research and educational use.
